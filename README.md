## Usage

### One-click install

Click on a script below to install it directly in your userscript manager (Greasemonkey or Violentmonkey):

| Script                                                                                                            | Description                                                     |
| ----------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------- |
| [GitLab Extras](https://github.com/knutkirkhorn/userscripts/raw/main/scripts/gitlab-extras.user.js)               | Extra shortcuts and improvements for GitLab                     |
| [ClickUp Timesheet Comma to Period](https://github.com/knutkirkhorn/userscripts/raw/main/scripts/clickup.user.js) | Automatically replaces commas with periods in time input fields |
| [GitLab Fix in Cursor](https://github.com/knutkirkhorn/userscripts/raw/main/scripts/gitlab-fix-in-cursor.user.js) | Add "Fix in Cursor" button to failed GitLab jobs                |

### Manual install

Copy the userscript from the [scripts](./scripts) folder to your browser's userscript extension.

## Development

Install dependencies for development. This is not required for using the scripts.

```bash
bun install
```
